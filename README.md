Retail Sales Analysis (SQL Project)
Project Overview

This project focuses on analyzing retail sales data using SQL.
The goal is to explore the dataset, clean the data, and answer important business questions using SQL queries.

The project demonstrates how SQL can be used for data exploration, data cleaning, and business analysis, which are essential skills for a data analyst.

Level: Beginner
Database: PostgreSQL
Database Name: p1_retail_db

Objectives

Create a retail sales database and table structure

Import and organize sales data

Perform data cleaning by identifying and removing null values

Explore the dataset to understand customer behavior

Use SQL queries to generate insights from the sales data
